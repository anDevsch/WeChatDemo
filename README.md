# 微信小程序开发

本仓库的目的是熟悉Wafer2框架而搭建的一个简单的小程序实例
本仓库基于最简版的Wafer2 Node.js Demo，引用WeUI.wxss模板，并实现了一个简单的json接口

## 开发环境

-  Node.js 8.2.1
-  MySQL 5.7.18
-  Nginx 1.10.2
